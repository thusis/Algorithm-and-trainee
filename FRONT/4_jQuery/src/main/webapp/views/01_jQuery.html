jQuery

존 레식에 의해 개발된 경량 JavaScript라이브러리
JavaScript코드를 쉽게 구현하기 위해 개발

jQuery 개요

jQuery장점

  1. DOM 과 관련된 처리를 쉽게 구현 가능
  2. Ajax통신, 이벤트 처리 등을 폭 넓게 지원
  3. 별도의 플러그인을 통해 차트, 슬라이드쇼, 테이블 등 간단히 구현 가능

jQuery연결

  1. jQuery 홈페이지에서 js파일을 다운로드 받아 연결
  2. CDN(Content Delivery Network)를 통한 연결

$(document).ready()

페이지를 로드한 후 ready메소드 실행 = JavaScript의 window.onload
하지만 자바스크립트의 window.onload 속성은 복수 개 설정 시 마지막에 쓴 것으로 덮어지지만
jQuery는 복수 개 설정하면 모두 연달아 실행됨
