스크립트 언어

기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도로 사용

자바스크립트

웹 브라우저에서 만힝 사용하는 인터프리터 방식의 객체지향 프로그래밍 언어

자바스크립트 사용법

1. 태그에 직접 간단한 소스 코드를 작성해서 실행하는 방법
2. script 태그에 작성하여 실행하는 방법
3. 별도의 js 파일로 작성해서 가져다 사용하는 방법

1️⃣태그에 직접 자바스크립트 코드를 작성해서 실행

2️⃣ script 태그 안에 함수 단위로 소스 작성하여 실행

3️⃣별도의 js 파일로 작성해 실행
