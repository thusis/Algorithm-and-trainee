폼 관련 태그

form 태그는 html에서 사용자가 입력할 수 있는 양식을 제공하는 태그로
form 태그 내의 input 태그들을 통해 사용자가 입력한 정보를 서버로 넘기는 역할을 함.
action 속성: 폼에 입력된 값들을 전송 받을 서버 입력



데이터가 전달되는 목적지(서버)를 명시하지 않으면 나 자신(현재 페이지)에게 데이터를 다시 보내게 되는데
이 때 (보낸 데이터를 포함한) 아예 새로운 페이지가 로딩된다.
그래서 URL에 "어쩌구 저쩌구..../7_폼태그관련.html?search=목요일" 이란 주소가 나오는 것이다.
위 개념은 서버부터 스프링파이널까지 널리 쓰이는 중요한 개념이므로 잘 알아두도록 하자

  • get 방식 : url 그대로 노출, default 값
  • post 방식 : url, 데이터 구조 노출하지 않는.
대체 왜 이렇게 졸린건데? 왜 잠이 전혀 안 깨다가 지금 깼다......
