기본 문법

변수와 자료형

1️⃣ 변수 선언

함수 내부에서 'var 변수명;' 으로 선언하면 지역변수가 됨
함수 내부에서 '변수명;'으로 선언하거나 함수 밖에서 '변수명;' 혹은 'var 변수명;'으로 선언하면 전역 변수가 됨
전역변수와 지역 변수가 동일한 이름인 경우 함수 내부에서 변수 명을 호출하면 지역 변수가 우선권을 가짐
전역 변수 사용 시 'window.변수명' 혹은 'this.변수명'으로 표현하여 지역변수와 구분
전역 변수는 해당 window내 어디든 사용할 수 있으며, 지역 변수는 해당 함수 내에서만 사용 가능

2️⃣ 자료형

자바스크립트에서는 자료형 별로 변수 타입이 지정되지 않고 리터럴에 의해 자료형 결정

3️⃣ typeof() 연산자

값의 자료형을 확인하는 연산자
선언 시 자료형을 지정하지 않기 때문에 변수 명만으로는 데이터의 자료형을 확인할 수 없음

4️⃣var 와 let 과 const

기존 ES5 버전에서는 모든 변수를 var 자료형으로 받아왔지만 데이터의 모호함이라는 문제를 개선하기 위해 최근 ES6 버전에서는 let과 const 를 사용하는 추세 -> 가독성 및 유지보수 증가
- let : 일반적인 변수 선언(var 와 같음, 덮어쓰기 가능)
- const : 상수 변수 선언(덮어쓰기 불가능)
