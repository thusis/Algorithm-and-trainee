BOM(Browser Object Model)

1️⃣ navigator 객체

웹 페이지를 실행하고 있는 브라우저에 대한 정보를 갖고 있는 객체

2️⃣ screen 객체

웹 브라우저 화면이 아닌 운영체제 화면(모니터)의 속성을 갖는 객체

availWidth / availHeight / width / height / pixelDepth / availLeft /
availTop / orientation / onchange / isExtended / addEventListener /
dispatchEvent / removeEventListener

3️⃣ location 객체

브라우저 주소 표시줄과 관련된 객체

[1] location.href

페이지 이동 : location.href='유알엘' 혹은 location='유알엘'

[2] reload()

페이지를 새로고침하는 메소드

  • location.reload() : 스크롤 위치 유지
  • location=location : 위로
  • location.href=location.href : 위로

[3] assign() 과 replace()

4️⃣ history 객체
