인덱스로 돌아가기

배열

자바스크립트에서는 자료형 지정이 없기 때문에
한 배열 안에 모든 자료형을 보관할 수 있다.



1️⃣ 배열 선언 및 초기화

배열 선언 시 크기를 지정하지 않고 선언할 수도 있고
크기를 정하고 선언할 수도 있다.



2️⃣ 배열 객체 메소드

array도 하나의 객체이기 때문에 배열에서 활용할 수 있는 메소드가 있음

[1] indexOf()

배열에서 요소가 위치한 인덱스 리턴

[2] concat()

여러 개의 배열 결합

[3] join()

배열을 결합하여 문자열로 반환

[4] reverse()

배열 순서를 뒤집음

[5] sort()

배열 정렬

[6] push()/pop()

push() : 배열의 맨 뒤에 요소 추가 / pop() : 배열 맨 뒤에 있는 요소를 리턴하면서 제거

[7] shift()/unshift()

shift() : 배열의 맨 앞의 요소를 리턴하며 제거 / unshift() : 배열의 맨 앞에 요소 추가

[8] slide()/splice()

sclie() : 배열의 요소 뽑아내기 / splice() : 배열의 index 위치의 요소 제거 및 추가

[9] toString()

배열을 문자열로 반환
